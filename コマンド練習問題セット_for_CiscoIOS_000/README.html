About "Command Practice".

Practice for your networks.

    PRACTICE QUESTIONS SET

Command Practice - Practice questions set.






"Command Practice - 練習問題セット"は、Ciscoルータ・スイッチのコマンドを覚えるため、 手を動かして様々なコマンドを覚えてもらう練習キットです。

本キットに含まれる"Command Practice"はシミュレータではないため、 設定変更による状態変化はありません。
あくまで、コマンド投入操作練習を繰り返すことでコマンドを覚える為のものです。




1.使用条件

●本セットの問題と同じ問題は何らかの試験には出題されません。 あくまで必要なスキルを身に着けるための練習道具です。

●本セットで練習したことで、何らかの試験に合格するのを保証するものではありません。

●本セットで採用している"Command Practice"ソフトウェアはオープンソースで公開されており再配布できますが、 本セットに含まれる問題集は再配布できません。 ただし、ご自身で"Command Practice"ソフトウェアを同梱した問題集を作成し配布して頂くことは可能です。

●本セットに含まれるものに起因する損害は補償いたしません。自己責任で利用願います。




2.使い方

番号の振られた各フォルダ内の「設問.html」を開き、設問の通り練習してください。
各フォルダ配下の装置名のフォルダ内の実行ファイルを開くと、コマンド入力画面が開きます。

また、解答は「解答.html」を確認してください。




3.権利関係

●Cisco IOSは、Cisco社が作成しているソフトウェアです。

●"Command Practice"は、DD-HOSTが作成しているソフトウェアです。
